3. [ ] 组织管理：部门/岗位/员工 CRUD

7. [ ] 公告管理：管理员发布 -> 员工查看 -> 未读数变化

## 常见问题

- 确保演示前清除测试数据中的敏感信息
- 密码和 Token 不在屏幕上完整展示

- 缺陷报告不能使用模糊描述
